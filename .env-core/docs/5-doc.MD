## 5.1 First setup

Run from the command line:

```
./setup.sh
```

First of all you need to run Nginx server [option 1]

| Step | Option | Description              |
|------|--------|--------------------------|
| 1    | 1      | Nginx                    |
| 2    | 1      | Setup Nginx server       |
| 3    | 0      | Return to main menu      |
